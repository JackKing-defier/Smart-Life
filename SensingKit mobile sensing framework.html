SensingKit

A Multi-Platform Mobile Sensing Framework for Large-Scale Experiments

WXApp

Multi-Platform

SensingKit works in both Android and iOS mobile platforms.

Multi-Sensory

It supports multiple sensors such as Motion, Orientation, Location, Activity and Proximity (using Bluetooth Smart).

Easily extensible

Due to its modular design, SensingKit is easily extensible.

Power efficient

It uses Bluetooth Smart for its proximity sensing features, having only a minimum impact on the device's battery life.

Automated Data Processing

SensingKit provides a plug-in system that allows developers to write custom data processing scripts in Python language.
(This feature is not available at the moment)

Open-Source

It is available in open-source under the GNU LGPL v3.0
